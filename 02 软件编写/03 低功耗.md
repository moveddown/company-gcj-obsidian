# 一、引脚注意点


- 未使用的引脚必须配置为模拟输入模式

- 进入低功耗前：
	1. 不用的普通 IO -> 模拟输入，无上下拉
	2. 需要保持状态的控制脚 -> 输出明确电平
	3. 唤醒脚 -> 保留输入/EXTI/上下拉
	4. 可能倒灌外设的通信脚 -> 模拟输入、高阻或输出低，按外部电路决定

- 唤醒后：
	1. 重新初始化 GPIO
	2. 重新初始化 UART/I2C/SPI/ADC 等外设
	3. 恢复业务运行

	**低功耗前统一切模拟输入，唤醒后重新初始化”是合理的大方向，但不能套到所有引脚。真正稳的做法是按引脚分类：未用脚模拟输入，唤醒脚保留唤醒配置，控制脚保持确定电平，[[01 电路设计/02 外设通信脚重点防倒灌|外设通信脚重点防倒灌]]**
