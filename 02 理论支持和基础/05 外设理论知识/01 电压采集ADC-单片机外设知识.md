
# 一、基础概念

### 1. 理解ADC

==ADC 不是凭空说“这个电压是多少V”。它真正输出的是一个**比例值**：==

`输入电压 占 参考电压 的比例`

比如 12 位 ADC 最大值是 4095，那么：

`ADC值 = 输入电压 / 参考电压 * 4095`


==**补充概念**==

**VDD**：MCU 的供电电压。  
比如电池直接给 MCU 供电，那电池现在是 3.8V，MCU 的 VDD 基本就是 3.8V。

**ADC输入**：你想让 ADC 去测的那个电压。  
比如你把 PA3 接到一个电池分压点，那 PA3 上的电压就是 ADC 输入。  
但是如果是使用内部电压采集，ADC 输入不是外部引脚，而是芯片内部的 1.2V 基准，那么ADC输入就是1.2V

**ADC参考电压**：ADC 用来当“满刻度”的电压。  
它决定 ADC 的最大量程。假设参考电压是 3.3V，那么 ADC 会认为：

`0V -> 0 1.65V -> 2048 左右 3.3V -> 4095`

所以 ADC 出来的 4095 不是“4095伏”，而是代表：输入电压已经等于参考电压

### 2. 基础ADC换算

``` text
假设
ADC参考电压 = 3.3V 
ADC输入电压 = 1.65V
```

12 位 ADC 输出：

`ADC值 = 1.65 / 3.3 * 4095 ≈ 2048`

这时候 ADC 的意思是：

`输入电压占参考电压的一半`

所以可以反算：

`输入电压 = ADC值 * 3.3 / 4095`

**这个就是电压采集的基础公式**
==**V_in = ADC_Value × V_ref / 2^N**==

### 3. 参考电压变动的ADC

上面公式可看出来有一个关键点，就是参考电压，一个是限制了量程的大小，一个是这个值不能变。

比如以下情况：
```text
假设电池直接给 MCU 供电：
VDD = 电池电压 ADC参考电压 = VDD

如果又拿 ADC 去采电池本身：
ADC输入 = 电池电压 ADC参考 = 电池电压

那 ADC 看到的是：
输入电压 / 参考电压 = 电池 / 电池 = 1

所以输出就是满量程：
ADC值 ≈ 4095
不管电池是 4.2V、3.8V、3.5V，只要 ADC 输入和 ADC 参考一起变，比例都接近 1，ADC 就分不出来。

```
关于上面的情况，就不能怎么做，应该使用内部电压作为参考电压
```text
ADC输入 = 内部固定 1.2V
ADC参考电压 = VDD = 电池电压

这时候 ADC 输出的是：
ADC值 = 1.2V / 电池电压 * 4095

如果电池是 4.2V：
ADC值 ≈ 1.2 / 4.2 * 4095 ≈ 1170
如果电池是 3.3V：
ADC值 ≈ 1.2 / 3.3 * 4095 ≈ 1489
电池下降后，ADC 值反而变大。因为 1.2V 在 3.3V 这个参考范围里占的比例更大了。

所以反推出：
电池电压 = 1.2V * 4095 / ADC值
```
**这个就是内部电压采集的基础公式**
==**V_in = 基准电压 *  2^N / ADC_Value **==

### 4.总结一下

ADC采集电压，其实就是 ADC的值=输入电压/参考电压 * 量程
如果参考电压不变，就是比如稳定的3.3V，就可以求出输入电压是多少，这个就是我们要求的电池采集的值
如果参考电压变，比如直接电池供电，电池电量低了，整体的单片机电流也就变化了，也就是VDD变化了，所以这个情况，可以用内容的基准电压1.2V作为输入电压，然后就可以退出参考电压。

==需要注意的是：因为数学公式的换算，可以看到不同方式ADC采集的值，对应的电流大小趋势是有差异的。一个是越大电流就越大，一个是越小电流就越小==


# 二、外部分压采集


根据上面来看，现在已经清楚了ADC采集的逻辑，但是因为参考电压的限制，还有不是所有情况都是电池直接接入单片机的。那么就有另外两种情况：
1. 分压电阻采集
2. 分压电阻结合内部基准电压采集

### 1. 分压电阻采集

```text
待测电压 Vin
   |
  R1
   |
   +---- ADC采样点 Vadc
   |
  R2
   |
  GND
  
R1 是上拉到待测电压的电阻
R2 是下拉到 GND 的电阻
ADC 接在 R1 和 R2 中间
```
通过上图分压计算公式，可得到 **Vadc = Vin * R2 / (R1 + R2)**
反推就可以得到公式为 ==**Vin = Vadc * (R1 + R2) / R2**==

然后在结合上面的Vadc采集可以得到公式为 ==**Vin=Vref * Vadc/量程 * (R1 + R2) / R2 ==


### 2.分压电阻结合内部基准电压

用内部的基准电压，也就跟上面的情况一样，就是参考电压不一定能保证不变，那么就可以结合上面的基准电压，加上分压电阻结合。通过这个也能看出分压电阻的和采集其实是各分其职的

```text
电阻分压：把高电压缩小到 ADC 能测
内部基准：校准/反推出 ADC 当前参考电压
```

显示内部反推ADC值： 
VDD = 1.2V * 4095 / ADC_bgr
这个就是 当前ADC参考电压=当前的VDD

```text
Vin
 |
 R1
 |
 +---- Vadc -> ADC
 |
 R2
 |
GND

ADC采集的分压点就是
Vadc = ADC_div * VDD / 4095
然后还原原始电压
Vin = Vadc * (R1 + R2) / R2
合起来就是 Vin = ADC_div * VDD / 4095 * (R1 + R2) / R2
上面就是 分压电阻采集
```

 其中又知道 VDD = 1.2V * 4095 / ADC_bgr 。带入到上面的式子，可得
 ==**Vin = ADC_div * 1200 / ADC_bgr * (R1 + R2) / R2**==

```text
举例：
内部基准 = 1.2V
R1 = 100k
R2 = 33k
ADC_bgr = 1365
ADC_div = 2048

先算VDD
VDD = 1200 * 4095 / 1365
VDD = 3600mV

在算分压点电压
Vadc = 2048 * 3600 / 4095
Vadc ≈ 1800mV

再还原输入电压
Vin = 1800 * (100 + 33) / 33
Vin ≈ 7255mV

```

**补充一下**
其中的ADC_div 和 ADC_bgr 都是 ADC 转换出来的“数字值”，不是电压本身。区别在于：**它们采的输入源不一样**

div就是采集外部的值，bgr是采集内部的值
```text
ADC_div：采外部电阻分压后的电压
ADC_bgr：采芯片内部固定基准电压
ADC_div 是“我要测的外部电压经过分压后的结果”
ADC_bgr 是“用来校准当前ADC参考电压的尺子”

假设
ADC参考电压 Vref = VDD = 3.6V
内部基准 Vbgr = 1.2V
外部分压点 Vdiv = 1.8V

那么
ADC_bgr = 1.2 / 3.6 * 4095 ≈ 1365
ADC_div = 1.8 / 3.6 * 4095 ≈ 2048

ADC_bgr 反映的是内部1.2V占VDD多少
ADC_div 反映的是外部分压点占VDD多少

所以先用 ADC_bgr 算当前参考电压：
Vref = Vbgr * 4095 / ADC_bgr
Vref = 1200 * 4095 / 1365
Vref = 3600mV
再用 ADC_div 算分压点电压：
Vdiv = ADC_div * Vref / 4095
Vdiv = 2048 * 3600 / 4095
Vdiv ≈ 1800mV

最后根据电阻分压比例还原外部电压：
Vin = Vdiv * (R1 + R2) / R2

所以
ADC_bgr 负责告诉你：当前ADC参考电压是多少
ADC_div 负责告诉你：外部分压点占当前参考电压多少

```
### 3. 总结一下


**使用内部基准电压**
先用内部已知电压校准 ADC 当前参考电压
再用校准后的参考电压计算外部采样电压
待测电压比 ADC 量程高，需要分压
MCU 的 ADC 参考电压又不是固定的
希望减少 VDD 波动带来的测量误差

**一般使用场景**
内部基准单独用：测 MCU 自己的 VDD
电阻分压单独用：参考电压固定时测高电压
电阻分压 + 内部基准：参考电压不固定时，测外部高电压更准
